# LWJGL-2.9-Game-Engine
A simple LWJGL 2.9 Game Engine based on ThinMatrix OpenGL tutorials.

# Features
* audio
* bloom effect
* camera
* light
* font mesh creator
* font rendering
* blur effect
* simple gui
* textured models
* object loader
* map effect
* particles
* post processing
* terrain renderer, height terrain generator
* entities renderer
* shaders
* shadow box
* mouse picker
* water
